# Personal Brand Site Aesthetic

## Objective
When designing or updating personal brand sites, use this "World-Class Ultra-Premium" aesthetic to ensure a sophisticated, authoritative, and incredibly polished look. 

## Core Principles
1. **The Abyssal Black Theme**:
   - Utilize a true, deep black background (`#030303` or `#050505`) rather than generic dark gray.
   - Contrast this tightly with pure white (`#FFFFFF`) or slightly muted white (`#A3A3A3`) text.
   - Avoid excessive colors. Pick one sophisticated focal accent (like a sharp gold `#D4AF37` or electric teal) but use it extremely sparingly (e.g., small highlight text, underline accents, or bullet points).

2. **Structural Elegance via Hairline Borders**:
   - Do NOT use heavy, rounded glassmorphic blobs.
   - Use sharp, strict grids (e.g., 2x2 symmetry) with very sheer borders (`1px solid rgba(255, 255, 255, 0.05)`).
   - Panels and cards should have `border-radius: 0px` (or practically 0) to evoke print editorial and luxury fashion/tech minimalism.

3. **Hyper-Scaled Typography**:
   - Typography should be aggressively scaled for impact. Section titles and hero text should dominate the negative space (`clamp(3.5rem, 8vw, 6.5rem)`).
   - **Contrast Fonts**: Use an elegant serif font (like `Newsreader`) for major headings paired with an ultra-clean sans-serif (like `DM Sans` or `Inter`) for body copy and UI elements (buttons, nav).
   - Use small, all-caps, tightly tracked `.eyebrow` text (`font-size: 0.75rem`, `letter-spacing: 0.2rem`) above major sections to define layout hierarchy cleanly.

4. **Strategic Layouts**:
   - Emphasize hard numbers and facts early. An "Impact Points" list near the hero section is mandatory.
   - Image integration should be seamless. Portraits should be masked (e.g. grayscale with a linear-gradient fade) so they blend directly into the dark background, avoiding rigid image boxes.
   - Use significant negative space (`padding: 120px 0`) to let the content breathe.

5. **Subtle Motion**:
   - Animations should be smooth, slow, and cubic (`cubic-bezier(0.16, 1, 0.3, 1)`).
   - Use slight upwards fades (`transform: translateY(40px)`) rather than aggressive bounces or bright flashes.
